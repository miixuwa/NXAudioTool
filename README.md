
# NXAudioTool

(Hopefully) The best NSW(2) audio player and converter for BFSTM, BFWAV, BFSAR, BFBNK, BFSEQ, and BWAV files.

Preview, organize and convert common Nintendo audio file types through a clean and lightweight UI built with Python and CustomTkinter.

---

# Features

- Play Nintendo audio formats directly
- Convert files to MP3 or WAV
- Drag & drop support
- Search functionality
- Volume and playback controls
- Auto-next playback
- Multi-file batch conversion

---

# Supported Formats

## Input

- BFSTM
- BFWAV
- BFSAR
- BFBNK
- BFSEQ
- BWAV

## Output

- MP3
- WAV

# Download

The Windows executeable can be downloaded instantly from this repository page without having to build anything yourself.

After installing, please un-zip the ZIP folder and keep everything inside it as it is.

To start the program, you only have to find the executeable inside the folder and run it, the rest should be pretty self-explanatory.
